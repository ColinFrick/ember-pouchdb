# PouchDB Wrapper for Ember.js




## Development

### Installation

1. npm install
2. bower install

### Run Tests

Terminal: ```grunt test```
Browser: 
1. ```grunt server``` 
2. Go to [http://localhost:8000/test](http://localhost:8000/test)

### Bump Version

1. grunt bump-only
2. grunt build
3. grunt test
4. grunt shell:git-add
5. grunt bump-commit
6. grunt release
